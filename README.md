# الماهر 📖

تطبيق متابعة حلقات حفظ القرآن الكريم — **جمعية الماهر بالقرآن وعلومه**.

## المزايا

- 🕌 **الحلقات**: مسجد البحر (الاثنين)، مسجد الياقوت، مسجد البحر (الأربعاء)، مسجد ابن تيمية — مع إمكانية إضافة حلقات جديدة.
- 🌸 **الطالبات**: لكل طالبة ثلاثة أهداف — هدف الحفظ 📖، هدف التثبيت 📌، هدف المراجعة 🔁 — مع علامة «تمّ» وملاحظات.
- 👩‍🏫 **المعلّمات**: كل معلّمة لها حلقاتها وطالباتها، وتدخل بياناتهنّ من صفحتها.
- 🏅 **لوحة الشرف**: مولّد إعلانات بهوية الجمعية — اختاري الحلقة والأسماء ويطلع لك إعلان جاهز للتحميل أو المشاركة صورةً.
- 💾 **نسخ احتياطي**: تصدير واستيراد البيانات بملف JSON.

## التقنية

- Next.js (App Router) + Tailwind CSS 4
- الخطوط: Reem Kufi للعناوين + Cairo للنصوص
- التخزين حالياً محلي على الجهاز (localStorage) — جاهز للترقية إلى Supabase لاحقاً

## التشغيل محلياً

```bash
npm install
npm run dev
```
